# 사주 리밸런서 도메인 지식

## 핵심 개념

- **사주(四柱)**: 년주·월주·일주·시주 — 태어난 시각의 우주적 기운을 4개 기둥으로 기호화
- **팔자(八字)**: 4기둥 × 2글자(천간+지지) = 8글자
- **일간(日干)**: 일주의 천간. 사주 분석의 핵심 — 그 사람의 본질적 기질
- **용신(用神)**: 일간을 도와주는 오행. 투자에선 재물을 키워주는 기운
- **기신(忌神)**: 일간에 해로운 오행. 투자에선 손실 유발 기운
- **재성(財星)**: 내가 극(剋)하는 오행 — 재물과 직결
- **오행(五行)**: 木·火·土·金·水 — 상생(木→火→土→金→水→木)과 상극(木→土→水→火→金→木) 관계

## CEO 사주 궁합 로직

### 흐름
```
티커 입력 → DB 캐시 조회 → (미스) DuckDuckGo 검색 → Gemini 파싱
→ CEO 생년월일 획득 → sajupy로 사주 계산 → Gemini 궁합 분석
```

### 타임존 원칙
- CEO 사주는 **출생지 현지 시간(Local Time)** 기준
- DuckDuckGo 검색 결과의 날짜를 그대로 사용 (KST 변환 없음)
- CEO 시진 불명 시 일간(日干) 위주 분석

### 궁합 점수 기준 (1~5★)
- **5★**: 일간 상생, 재성 왕성, 운세 흐름 최적
- **4★**: 대체로 긍정적, 단기 흐름 양호
- **3★**: 보통, 중립적 관계
- **2★**: 상극 관계 있음, 주의 필요
- **1★**: 강한 상극, 투자 비권장

### 추천 기준
- **매수**: 점수 4~5
- **관망**: 점수 3
- **주의**: 점수 1~2

## 포트폴리오 리밸런싱

### 통화 처리
- `currency: "KRW"` — 한국 주식, 원화 기준
- `currency: "USD"` — 미국 주식, 달러 기준
- `current_value`는 **항상 KRW** (USD 종목은 × 1400 환산)

### 액션 레이블
- 일반 종목: 매수 / 매도 / 유지
- 현금 항목: 비중확대(매수) / 비중축소(매도) / 유지

## 데이터베이스

### SajuCache
- 키: `(birth_date, birth_hour, gender)`
- TTL: 당일(KST) — 대운/세운은 날짜 기반

### CeoCache
- 키: `ticker` (대문자)
- TTL: 없음 (CEO는 잘 안 바뀜)
- 잘못된 정보 신고 → `CeoFeedback` 테이블

### RebalancingReport
- UUID 기반 공유 링크
